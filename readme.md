a simple minimalist study dashboard.

- Pure client-side: HTML, CSS, and JavaScript (no backend)
- Features: task list; Pomodoro timer; and more to come.
- How to run locally: open index.html in a modern browser or serve via a static server (for example, npx serve)
- Compatibility: latest Chrome, Edge, and Firefox
- Status: work in progress; feedback and contributions welcome
